## Basic operations
- Insert mode : `i`  
- Append mode : `a`
- Press `Esc` to exit from insert mode  
- Save : `:w`  
- Save and quit : `:wq`  
- Quit without saving : `:q!`  

## Advanced navigation
- Move to next word : `w`  
- Move to previous word : `b`  
- Start of line : `0`  
- End of line : `$`  
- Jump to last line : `G`

## Copy pasting
- Start/stop selection mode : `V`  
- Copy selection : `y`  
- Paste : `P`
